## how to use
1. download the script folder
2. fusion 360 → **Tools → Add-Ins → Scripts and Add-Ins** → click **"+"** → select the folder
3. hit **Run**

> save your file first. and turn off design history before running anything on big assemblies (right-click root → *Do Not Capture Design History*)

## notes
- uses `adsk.doEvents()` so fusion doesn't freeze mid-script
